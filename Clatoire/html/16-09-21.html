Classe du 16/09/21 [CIN1]

HTML

Tag

Les tags sont ce qui permet d'organiser notre contenu sur la page.
<h1> Tags </h1> Afficherais Tags en grand. h2 l'aurais afficher un peu plus petit.

Meta tag

Un metatag est un tag donnant des infos sur la page.
Par example il definit l'encodage de la page : <meta charset="UTF-8">

Element

Un element est tout le contenu entre les tags ouvrant et fermant.
eg. <h1 class="title"> Classe du 16/09/21 [CIN1] </h1> - ici l'element c'est "Classe du 16/09/21 [CIN1]"

CSS

Avec le css on peut redefinir la couleur du texte et du fond
En le combinant aux pseudos-classes, on peut changer ces couleurs sur un etat d'un element
(comme pour le lien ci dessous - On change son fond et sa couleur lorsqu'on passe la souris dessus)
